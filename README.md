-Descrição
Este projeto é uma calculadora básica desenvolvida em linguagem C.
Ela permite que o usuário realize as seguintes operações matemáticas:
•	Soma
•	Subtração
•	Multiplicação
•	Divisão
O programa exibe um menu interativo, aceita duas entradas numéricas do usuário e mostra o resultado da operação escolhida. A calculadora continua em execução até o usuário optar por sair.
-Funcionalidades
•	Menu interativo com 5 opções.
•	Funções separadas para cada operação matemática.
•	Loop contínuo até que o usuário escolha sair.
•	Tratamento para evitar divisão por zero.
•	Organização de código com boa estrutura e legibilidade.
-Tecnologias utilizadas
•	Linguagem C (C99)
•	Compilador: GCC (ou qualquer outro que suporte C padrão)
